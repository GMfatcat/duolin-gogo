## zh-TW

HTTP header 常用來描述內容型態、認證資訊、快取策略、壓縮方式等附加資訊。
它不是主要資料本體，但會強烈影響伺服器和客戶端如何理解這次請求或回應。

## en

HTTP headers commonly describe extra information such as content type, auth data, caching rules, or compression behavior.
They are not the main payload, but they strongly influence how the client and server interpret the request or response.
