## zh-TW

HTTP status code 通常可以先看第一位數字理解大類，例如 2xx 常代表成功、3xx 代表重新導向、4xx 代表用戶端問題、5xx 代表伺服器問題。
這是一個很實用的快速判讀方式。

## en

HTTP status codes can often be understood first by their leading digit: 2xx usually means success, 3xx redirect, 4xx client-side problem, and 5xx server-side problem.
That is a practical way to classify a response quickly.
