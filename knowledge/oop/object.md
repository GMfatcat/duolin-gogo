## zh-TW

如果 class 是一份規格，那 object 就像依照這份規格產生出來的實體。
不同 object 可以來自同一個 class，但各自有自己的資料狀態。

## en

If a class is a specification, an object is a real instance created from it.
Different objects can come from the same class while still holding their own state.
