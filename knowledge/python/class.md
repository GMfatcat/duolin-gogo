## zh-TW

Python 的 class 常用來定義某類物件應該有哪些資料和方法。
依照 class 建立出來的個體就是 object。
當程式需要把狀態與行為綁在一起時，class 很常出場。

## en

A Python class is often used to define what data and methods a kind of object should have.
An individual created from that class is an object.
Classes are useful when state and behavior should live together.
