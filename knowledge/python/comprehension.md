## zh-TW

Python comprehension 讓你用比較精簡的寫法建立新集合。
它常見於 list comprehension，也有 set 和 dict 的版本。
核心概念是把迴圈、篩選、收集結果這幾件事濃縮在一起。

## en

Python comprehensions let you build new collections with a compact syntax.
They are common for lists, and there are also set and dict variants.
The key idea is combining looping, filtering, and collecting into one expression.
